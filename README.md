# 🍽️ खाना हिसाब - Khana Hisab

Food Hotel Company Management System with multi-language support.

## Features
- 📊 Dashboard with real-time stats
- 💰 Hisaab Kitaab (Transactions)
- 🍽️ Menu Manager
- 🛒 Order Manager
- 📈 Reports & Analysis
- 🌍 Hindi / English / Marathi

## Deploy Free on Render
1. Push to GitHub ✅
2. Go to render.com → New → Blueprint
3. Connect this repo → Auto-deploy!

## Local Dev
```bash
npm install
npx prisma generate
npm run dev
```
